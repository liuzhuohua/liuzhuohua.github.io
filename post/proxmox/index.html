👏 分类 Proxmox ！

✍️ Proxmox Proxmox VE是一款套开源的虚拟化管理软件，用户可通过网页的方式来管理服务器上使用 kvm 以及 lxc 技术运行的虚拟机。同时提供了一些先进功能的支持，如集群、HA等。

需要用到的命令👇

📝 一：rm /etc/apt/sources.list.d/pve-enterprise.list

📝 二：wget http://mirrors.ustc.edu.cn/proxmox/debian/proxmox-ve-release-5.x.gpg -O /etc/apt/trusted.gpg.d/proxmox-ve-release-5.x.gpg

📝 三：wget http://mirrors.ustc.edu.cn/proxmox/debian/proxmox-ve-release-6.x.gpg -O /etc/apt/trusted.gpg.d/proxmox-ve-release-6.x.gpg

📝 四：echo 'deb http://mirrors.ustc.edu.cn/proxmox/debian buster pve-no-subscription'>/etc/apt/sources.list.d/pve-install-repo.list

📝 五：sed -i 's/ftp.debian.org/mirrors.163.com/g' /etc/apt/sources.list

📝 六：sed -i 's/security.debian.org/mirrors.163.com/debian-security/g' /etc/apt/sources.list
